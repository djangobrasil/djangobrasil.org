Qual versão é melhor?

O Django é melhorado quase todos os dias e seus desenvolvedores utilizam da boa prática de manter o código estável. Dessa maneira, é seguro utilizar o último código de desenvolvimento e você ainda poderá se beneficiar das últimas funcionalidades implementadas. Apenas tenha em mente que ocasionalmente poderá ocorrer mudanças que torne o código de seu projeto incompatível com o novo código do Django, sendo necessário algumas adaptações. Para acompanhar as mudanças, dê uma atenção especial à linha do tempo de desenvolvimento.

Se você estiver apenas procurando uma distribuição estável e não se importar em aguardar a próxima versão, voce vai querer a última versão oficial (que incluem sempre notas detalhadas sobre as eventuais alterações que você precisará fazer ao atualizar)
